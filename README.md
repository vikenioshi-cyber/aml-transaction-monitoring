# AML & Transaction Monitoring System

## 📌 Overview
This project demonstrates an automated **Anti-Money Laundering (AML)** monitoring system. As an Economist with an auditing background (**PwC**), I designed this tool to identify suspicious financial patterns using Python, focusing on regulatory compliance (COAF/BCB standards).

## 🚀 Key Features
- **Anomaly Detection:** Identifies transactions that deviate significantly from a customer's declared income.
- **Risk Scoring:** Implements a weighted scoring system based on transaction amount, merchant category, and income discrepancy.
- **Regulatory Thresholds:** Automatically flags transactions exceeding R$ 10,000.00, aligning with local reporting standards.

## 🛠 Tech Stack
- **Python:** Pandas, NumPy.
- **AI Integration:** Prompt Engineering was used to optimize the data filtering logic and script efficiency.
